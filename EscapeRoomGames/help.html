Short Cut Keys


Control-Q {Enter command line mode}


  show clue01 {show clue directory/file without extension}

  play sound01 {play sound file sound01.wav}

  video video01 {play video file video01.mp4}


Control-S {Start timer}

Control-E {End Game / Stop timer / take screen shot}

Control-P {Pause timer}

Control-O {Pause timer}

Control-R {Resume timer}

Control-T {Reset timer}


Control-C {Clear message}

Control-D {Clear message}

Control-M {Set message}


Control-Plus {Add 60 seconds to timer}

Control-Minus {Remove 60 seconds from timer}


Control-H {Hide/Show game tree}
